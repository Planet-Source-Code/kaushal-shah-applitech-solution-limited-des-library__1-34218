DES Library 1.1 - DEMO Version

DES Library 1.1 is a VB component which can be used with any VB or ASP application to add cryptographic capabilities. Using this component, DES or 3DES encryption and decryption can be done.

Following example shows how to use this component in an VB or ASP document.

Put the file ASLDES.dll into your system directory (Ex. c:\winnt\system32 for WinNT or Win2K systems). Register this dll by typing the following command at the command prompt:

C:\>regsvr32 c:\winnt\system32\ASLDES.dll

To use DES Library in a VB environment, open a VB project, go to Project/References... and check the box next to DES_LIBRARY.  Declare an DES Library object variable as follows:

Dim crypto as clsEncrypt

Create an instance of the DES Library object as follows:

Set crypto = New clsEncrypt

To use DES Library in an ASP environment, you must create an instance of the DES Library object in your ASP script as follows:

<%
...
Set crypto = Server.CreateObject("DES_Library.clsEncrypt")
...
%>

To perform DES encryption, do the followings:

crypto.inputString = VariableInputString
crypto.keyDES = VariableDESKey
crypto.mode = mode_ENCRYPT 
VariableEncryptedOutput = crypto.encryptDES

To perform DES decryption, do the followings:

crypto.inputString = VariableInputString
crypto.keyDES = VariableDESKey
crypto.mode = mode_DECRYPT
VariableDecryptedOutput = crypto.encryptDES

To perform 3DES encryption, do the followings:

crypto.inputString = VariableInputString
crypto.key3DES1 = Variable3DESKey1
crypto.key3DES2 = Variable3DESKey2
crypto.mode = mode_ENCRYPT
VariableEncryptedOutput = crypto.encrypt3DES

To perform 3DES decryption, do the followings:

crypto.inputString = VariableInputString
crypto.key3DES1 = Variable3DESKey1
crypto.key3DES2 = Variable3DESKey2
crypto.mode = mode_DECRYPT
VariableDecryptedOutput = crypto.encrypt3DES
